# Simple Calculator in Python

## Description
This is a simple calculator project built using Python. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division. This project is created for learning and practicing Python fundamentals.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

## Technologies Used
- Python

## How to Run
1. Install Python in your system.
2. Download or clone this repository.
3. Open terminal or command prompt.
4. Run the file using the command:
   python_calculator.py

## Example
Enter first number: 10

Enter second number: 5

Choose operation: + - * /
  
Result: 10 + 5 = 15

## Purpose
This project is made for beginners to understand basic Python concepts like:
- Input/Output
- Conditional statements
- Basic arithmetic operations
